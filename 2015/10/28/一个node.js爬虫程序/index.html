一个node.js爬虫程序

10月 28, 2015

通过一段很短的node.js代码，就可以实现一个小的爬虫程序。
创建crawler.js程序：

var http = require('http')
var url = 'http://www.imooc.com/learn/348'

http.get(url,function(res){
    var html = ''

    res.on('data',function(data){
        html += data
    })

    res.on('end', function(){
        console.log(html)
    })
}).on('error', function(){
    console.log('获取课程数据出错！')
})

启动程序：

node crawler.js

成功地爬到了慕课网的课程信息。
